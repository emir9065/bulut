# Bulut Biome Combat Platformer

A side-view biome action platformer built with Vite + JavaScript.

## Run

```bash
npm install
npm run dev
```

## Controls

- Arrow keys: Move
- `A`: Attack
- `Shift`: Dash
- `Esc` / `P`: Pause
- `F`: Toggle fullscreen (`Esc` exits fullscreen)
- `K`: Open shop (after unlock)

## Campaign Biomes

1. Forest
2. Sea
3. Volcano
4. Snow
5. Cave
6. Space

## Testing Hooks

- `window.render_game_to_text()`
- `window.advanceTime(ms)`
