# Bulut Biome Combat Platformer

A side-view biome action platformer built with Vite + JavaScript.

## Run

```bash
npm install
npm run dev
```

## Controls

- Arrow keys: Move
- `A`: Attack
- `Shift`: Dash
- `Esc` / `P`: Pause
- `K`: Open shop (after unlock)

Mobile:
- Fullscreen web layout with overlaid touch controls
- Left thumb movement pad + right action buttons (`JUMP`, `ATK`, `DASH`)
- Canvas shortcuts: left-half tap = jump, right-half tap = attack, right-half double-tap = dash

## Campaign Biomes

1. Forest
2. Sea
3. Volcano
4. Snow
5. Cave
6. Space

## Testing Hooks

- `window.render_game_to_text()`
- `window.advanceTime(ms)`
